# _FORMAT — One File. Schema Included.

_FORMAT is an open specification for structured Markdown where every file carries its own schema. Define concepts, relationships, and evaluations in plain text. Drop it into any AI model or open it in the visual editor. Templates available for business modeling and procedures — or define your own.

- [Documentation](https://format.innv0.com/documentation/)

- [About](https://format.innv0.com/about)

## How it works

1. **Template** — Pick a schema: Business (stakeholders, strategy, finance), Procedures (workflows, roles, steps), or define your own concepts and matrices.
2. **Model** — Declare concepts, elements, and their fields in structured Markdown blocks. Every file is self-describing.
3. **Map** — Define relationships between concepts using matrix tables. The app visualizes these connections automatically.
4. **Validate** — AI scores the coherence of your entire model, highlighting gaps and suggesting improvements.
5. **Use** — Drop the file into ChatGPT, Claude, or Gemini. Commit it to Git. Share it with your team.
