What makes _FORMAT different

A single, portable Markdown file that humans read, AI understands, and Git versions.

📄

Plain Markdown

Author in any text editor. Review on GitHub. The format prioritizes both human readability and machine processing — no proprietary tools required.

🔗

Built-in Relations

Define matrix tables that map how concepts and elements interconnect — stakeholders to segments, procedures to roles, or any relationship your domain needs.

🤖

AI-Native

Drop your model into ChatGPT, Claude, or Gemini. The AI instantly understands the full context of your model without extra prompting or fine-tuning.

🧩

Template System

Choose from official templates (Business, Procedures) or define your own schema. Every file is self-describing — the template lives in the frontmatter.

📊

Visual Editor

📦

Version-Controlled

How it works

1

Pick a template

Choose from official templates (Business, Procedures) or define your own schema from scratch. Every model carries its template in its frontmatter.

2

Model

Define concepts, elements, and their fields in structured Markdown blocks. Each concept is a section; each element is a list item with optional YAML fields.

3

Map

Define relationships between concepts using matrix tables. The app visualizes these connections as interactive graphs and scored matrices.

4

Validate

The app scores the coherence of your model, highlighting gaps and suggesting improvements using the template's built-in markers.

5

Use

Drop your file into any AI model, commit it to Git, or share it with stakeholders. Your entire model fits in one portable, self-describing file.
