# About _FORMAT

_FORMAT stands for **Flat, Omnipotent, Readable, Model, Annotated, Template**.

It is an open-source, human-readable, agent-friendly format for representing business models in a single Markdown file. It is designed to be easily authored by humans, generated and analyzed by AI agents, and versioned cleanly in Git.

## Why

Traditional business modeling tools store data in opaque formats. This creates problems:

- **No portability** — models can't be shared without dedicated software
- **Poor diffing** — database exports produce noisy diffs
- **AI friction** — LLMs struggle with complex relational structures

_FORMAT solves these by representing business models as structured Markdown.
