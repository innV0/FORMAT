# _FORMAT

**Flat, Omnipotent, Readable, Model, Annotated, Template**

## One file. Your entire business — finally ready for AI.

_FORMAT reads a single structured `.md` file that holds your complete business strategy: segments, problems, value propositions, roadmap, finances, team — all interconnected. Drop it as context into any AI model and get strategic answers. Commit it to Git and get a living, visual, version-controlled business model.

## How it works

1. **Connect** — Create a _FORMAT file or clone an existing one.
2. **Model** — Declare stakeholders, problems, and value propositions in structured Markdown.
3. **Map** — Define relationships between concepts using matrix tables.
4. **Validate** — AI scores the coherence of your entire model.
5. **Use** — Drop the file into ChatGPT, Claude, Gemini — the AI instantly knows your business.
